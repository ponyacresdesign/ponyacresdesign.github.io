HTML Structure

The HTML structure heavily based on HTML structure of the Bootstrap Front end template.

Our responsive HTML structure have been developed from 1170px grid layout of Bootstrap Framework. HTML structure for specific section can be found below.

 Header

Akaposs’s header html includes

  • A big background image that is represented by img tag.
  • Floating menu that is at the bottom of header and will be a sticky menu when user scroll to other section. The html for this floating menu is based on navbar of Bootstrap.
  • Position absolute h2 element to hold the floating text. This text is wrapped in a container div of bootstrap to provide it sufficient margin.

Refer to header section in index.html for more information

Service section (Feature)

Akaposs’ service section developed based on bootstrap grid. The margin is provided by class container of Bootstrap. The detail implementation includes

  • h2 tag to hold the section title
  • Group of 4 column default item grid. Each of the item support
    • h4 tag for title
    • Icon using i tag and font awesome
    • Sepration line using div
    • p tag to hold the description

Refer to service section on index.html for more information

Emphasize section ( Business promote )

Aksposs’s emphaisze section currently developed to support an impression text message on an background image. This section’s html structure includes:

  • Full width background image that will be positioned to center. The static version must be held inside an img tag. For parallax use it as a background image.
  • For static version we support different image for mobile and desktop. Using hidden-xs, visible-xs to develop this.
  • div with class center-text to hold the text message.

Refer to Emphasize section index.html for more information

Experience section

Akaposs’s experience section had been developed using container and row class of bootstrap to provide adequate margin support. The detail HTML structure included.

  • Multile slides using the same structure as flexslider.
  • Each slide consist of:
    • div to hold the block of house image.
    • div for main text
    • div to hold the year time
  • Each list dot is developed using font awesome.

Refer to Experience’s section index.html for more information

Skill section

Akaposs’s skill section also use the same margin that had been provided with bootstrap container and row class. The HTML’s detail consist of:

  • Background image to function as parallax background image.
  • List of div item as holder for the text and the circle. This list item position followed Bootstrap grid.
  • Each list have following html structure
    • Default span tag that is compatible with Stellar html structure.
    • span tag to hold the text.
  • Further html like canvas will be rendered when Stellar plugin had been applied

Refer to Skill’s section index.html for more information

Portfolio section

Akaposs’s skill section also use the same margin that had been provided with bootstrap container and row class. The HTML’s structure is compatible with require structure for Quicksand plugin and pretty photo plugin. The details is as below:

  • h2 tag for the section’s title.
  • ul structure with filter value as attribute for li tag. This ul structure is for quicksan plugin.
  • Another ul tag to hold all the project.
  • Each project represented as a li tag with position class based on bootstrap grid.
  • Each project item consist of:
    • div class title for holding the project’s title
    • div with class excerpt to hold project excerpt
    • Zoom icon from font awesome.
    • a tag with compatible html structure for pretty photo plugin

Refer to Portfolio’ssection index.html for more information

Testimonial section

Akaposs’s skill section also use the same margin that had been provided with bootstrap container and row class. The HTML’s structure is compatible with flexslider plugin. The details is as below:

  • Flexslider’s html structure to prove slide function
  • Each flexslider slides consist of multiple item.
  • Each item have the same html as follow:
    • div with “voice” class to hold the content of testimonial
    • div with “position” class to hold further information of the testimonial’s author.

Refer to Portfolio’ssection index.html for more information

Contact section

Akaposs’s contact section also use the same margin that had been provided with bootstrap container and row class. This section included three widget

Socialize’s widget

Socialeize’s widget take 1/3 column of the contact section. The html of this widget includes:

  • p tag with class “title” to hold the widget’s title
  • ul tag with each li as a holder for social’s icon.
  • Icon for each of the li is font awesome.
  • mutltile p tag to hold phone, adress, email

Refer to Socialize’s widget in index.html for more information

Send pigeon widget

Send pigeon widget consist of:

  • Input text for name
  • Input text for email
  • Input text for message.

Refer to Send a pigeon widget in index.html for more information

Recent project widget

recent project widget consist of:

  • p with class “title” for widget title
  • ul tag to hold a list of project.
  • Each project consist of small img to represent it.

Refer to recent project widget in index.html for more information

Footer section

Akaposs’ footer section using a grid of 5/7 from bootstrap to provide:

  • div column to hold the copyright information
  • div column to hold the link to each section of the page

Refer to Footer’s section in index.html for more information
