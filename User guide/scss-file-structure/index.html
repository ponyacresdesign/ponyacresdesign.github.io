SCSS File & Structure

Akaposs’s template had been developed with modular code in mind. Therefore you will find multiple scss file inside scss folder. Each file is responsive for a section or a specific concern. All of them would be included inside the style.scss.

Common rule style

All the common rule for our template are also divided to specific file. You can find those information in _common.scss file. Those common style include:

_text.scss

This file includes all the style related to font such as font-size, font-weight, font -family for all section. Therefore you won’t need to go to different file to customize the font. Just one place like that.

_animate.scss

This file includes all the style related to Css3 animition.

_flexslider.scss

This file includes all the style related to flexslider. All the rewrite style we need to adapt the flexslider for out template.

_parllax.scss

This file includes all the style related to parallax effect.

_section.scss

This file includes all the common style for section.

We are also have specific file for each section

Normal section file

Each section file below contains the style that control how that section look, the detail position of each element. The mobile and desktop style of each section all contained in same file.

_header.scss

This file contains the style related to the header.

_menu.scss

This file contains the style related to the menu.

_feature.scss

This file contain the style related to the Service section.

_emphasize.scss

This file contains the style related to the Business promote section.

_experience.scss

This file contains the style related to the Resume section.

_skill.scss

This file contains the style related to the Skills section.

_portfolio.scss

This file contains the style related to the Projects  section.

_testimonial.scss

This file contains the style related to the Testimonials section.

_widget_contact.scss

This file contains the style related to the Contact form widget.

_widget_info.scss

This file contains the style related to the Social widget.

_widget_recent_proj.scss

This file contains the style related to the Recent project widget.

Because Bootstrap does not fit to all the things. Therefore we also provide

_reset.scss

This is for reseting all the style that we need.

Specific customization

We are supporting parallax. Therefore it’s very convenience to provide one place we you can just change the most important value of parallax section – the image background. You can now change image background of the parallax at file _specific.scss
